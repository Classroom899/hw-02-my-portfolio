Goals Calendar

Our second group project was a goals calendar app that will track the user's goals and progression made on those goals. Each short-term goal or long-term goal will be added into whatever date selected. The user can watch their progess towards reaching said goal. Goals can also be deleted. Daily motivational quote included because who doesn't need a little motivation here and there?

Technologies Used:
  • MySQL Workbench
  • Sequelize
  • Open quotes API
  • ESLint/Travis
  • Javascript/jQuery
  • HTML/Handlebars
  • CSS
  • Deployed via Heroku
